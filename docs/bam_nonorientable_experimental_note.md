Möbius_c = 3135 MeV, Möbius_b = 6468 MeV. Decay = twist-unwinding (non-orientable → orientable; sheds 2√σ as light isoscalar hadrons).

## 3. Preferred / suppressed modes (hybrid selection rule)

- **SUPPRESSED** — Möbius baryon → (ground heavy baryon) + (single S-wave π) (the naive, phase-space-favored channel).
- **PREFERRED** — Σ_Q π (spin-1 light diquark); Λ_Q (ππ)_{S, isoscalar} — high m(ππ); P-wave heavy baryon + π.
- An ordinary **radial excitation** does the opposite (single π to ground): the branching **pattern** is the discriminator.

## 4. Analysis handles
