# Geometric origin of `M_R` from throat-nucleation compliance (PR #87)

Follows the neutrino-quadrant suppression probe (PR #86), which
identified the neutrino as **Majorana** (because `k=0 ⟹ c₁=0 ⟹
C-invariant`) and the suppression as a **Majorana seesaw**
`m_ν = m_D²/M_R`, but left the seesaw scale `M_R ≈ 0.3–1.8 TeV` as an
**open** heavy input — no scale in the BAM catalog lands at `~TeV`.
This probe tests whether `M_R` is generated by the PR #58 throat-
nucleation barrier / boundary-compliance physics.

## The `ΔL=2` mechanism IS the PR #58 throat ↔ antithroat transition

A Majorana mass connects a state to its own antiparticle. In BAM the
antiparticle of a throat is the **antithroat** — the inner/outer swap
`C` (`c₁ → −c₁`, PR #63), the antipodal `Z₂` whose vacuum nucleation
PR #58 analysed (`E_nuc(R) = 4πσR² − (4/3)πρR³`, critical radius
`R_c = 2σ/ρ`, barrier `E_c = (16π/3)σ³/ρ²`). So a Majorana mass is
literally a throat ↔ antithroat transition, and the PR #58 nucleation
framework is its natural home.

## The selection rule: re-deriving PR #86 from PR #58's `Σc₁=0`

PR #58's conservation law `Σc₁=0` (one Hopf charge per throat; vacuum
`c₁=0`) applied to a **single** state flipping throat ↔ antithroat is
*exactly* PR #86's only-neutrino selection rule:

| sector | `k` | `c₁` | `ΔΣc₁` (single flip) | single-state Majorana? | nature |
|---|---:|---:|---:|:---:|---|
| neutrino | 0 | 0 | `0 → −0 = 0` | ✓ allowed | **Majorana** |
| charged lepton | `2g−1` | ±1 | `+1 → −1 ⟹ ΔΣ=∓2` | ✗ forbidden | Dirac |

Only `k=0` keeps `Σc₁=0` with no partner, so only the neutrino gets a
single-state `ΔL=2` Majorana mass; a charged lepton would need a pair
(the PR #58 C-conjugate pair), so it stays Dirac. PR #86's "only the
`c₁=0` sector is Majorana" **is** PR #58's `Σc₁=0` for a single state.
The two PRs are one statement.

## Hypothesis A — `M_R` = nucleation barrier height (FALSIFIED on scale)

The most direct reading: `M_R` is the barrier height
`E_c = (16π/3)σ³/ρ²`. Fix the brane tension `σ` and bag density `ρ`
from the **electron** throat (geometric units `m_e c² = 1`, `R* = 1`):

  - `B = 4πσ` with `E(R*) = 3BR*² = m_e c²` ⟹ `σ R*² = m_e c²/(12π)`,
    i.e. `σ = 1/(12π) ≈ 0.0265`.
  - bag density `ρ` = rest energy over the throat volume
    `(4/3)πR*³` ⟹ `ρ R*³ = 3 m_e c²/(4π)`, i.e. `ρ = 3/(4π) ≈ 0.239`.

Then

```
R_c = 2σ/ρ ≈ 0.22 R*,    E_c = (16π/3)σ³/ρ² ≈ 0.0055 m_e c² ≈ 2.8 keV.
```

That is `~10⁸–10⁹` **below** the required `~TeV M_R`. The soft vacuum
bubble of PR #58 is the **maximally compliant** throat-opening channel;
its barrier height cannot be the heavy Majorana scale, and BAM has no
stiff `~TeV` barrier or partner state. A *static* barrier-height /
compliance origin of `M_R` is excluded. **Hypothesis A fails.**

## Hypothesis B — suppression = throat ↔ antithroat tunnelling (VIABLE)

The smallness need not come from a large `M_R` at all. PR #58 flagged
the tunnelling/instanton rate as its open follow-on. If the Majorana
(throat ↔ antithroat) mass is the tunnelling **amplitude through** the
barrier rather than its height, then

```
m_ν = m_D · e^{−S},
```

with `S` the Euclidean bounce action for the `ΔL=2` flip. The required
action is a data-derived, dimensionless number:

| gen | `m_D` (keV) | `m_ν` obs (eV) | `S = ln(m_D/m_ν)` | `M_R = m_D·e^{S}` (GeV) |
|---:|---:|---:|---:|---:|
| 1 | 42.9 | 0.001 | 17.6 | 1836 |
| 2 | 80.2 | 0.009 | 16.0 | 715 |
| 3 | 117.6 | 0.05 | 14.7 | 277 |

`S ≈ 15–18` — modest, `O(15)`, and generation-stable. The seesaw
relation becomes

```
M_R = m_D²/m_ν = m_D · e^{S},
```

so the `~TeV` scale is **not** a fundamental heavy mass — it is the
`keV` Dirac floor (the cavity zero-point) **exponentially lifted** by
the throat ↔ antithroat tunnelling action. The open input is no longer
a mysterious `TeV` scale (6 orders above any BAM scale, with no
candidate state) but a modest `O(15)` bounce action — the very
instanton number PR #58 left open.
