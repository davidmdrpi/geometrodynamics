## The handle

Preferred channel `Λ_Q(ππ)_S` (twist-unwinding). **Dipion endpoint** `m(ππ)_max = M_Möbius − M_ground = 2√σ ≈ 849 MeV — flavor-independent** (same above charm and bottom), spectrum peaking high. A fixed edge in a directly-plotted observable; one overlay tests the framework.
