# Möbius throat exchange-sign probe — research plan

Follow-on to PR #43 (Dirac-trace geometry). PR #43 showed that
BAM's SU(2) Hopf-bundle spinor structure (Pauli matrices) reproduces
all QED 4-fermion **diagonal-channel numerator** structures
`(s²+u²)`, `(u²+t²)`, `(s²+t²)` for Bhabha and Møller. The only
remaining QED ingredient is the **relative interference sign** —
the `-1` between Bhabha s-channel and t-channel diagrams (from
Wick contraction) and between Møller t-channel and u-channel
diagrams (from Pauli antisymmetrisation of identical electrons).

This probe tests whether that sign emerges automatically from
BAM's `T = iσ_y` non-orientable antipodal throat transport (the
README channel-2 spinor structure derived in
`geometrodynamics.embedding.transport.derive_throat_transport`).

## The Möbius / ε structure

The BAM throat-transport matrix derived from the orientation-
reversing Hopf-preserving map on `S³`:

```
T  =  i·σ_y  =  [[ 0,  1 ],
                 [−1,  0 ]]
```

is **exactly** the SU(2) antisymmetric ε tensor:

```
ε_{ab}  =  T_{ab}
```

This is the natural symplectic / antisymmetric form on 2-D
spinor space. Two-spinor states decompose into:

  - **Triplet (s = 1)**: symmetric, 3-dimensional, exchange
    eigenvalue **+1**.
  - **Singlet (s = 0)**: antisymmetric, 1-dimensional, spanned
    by `ε_{ab}·ψ^a·χ^b = ψ^↑ χ^↓ − ψ^↓ χ^↑`, exchange
    eigenvalue **−1**.

The fermion (anti-commuting) two-particle state lives in the
antisymmetric singlet sector. Exchange of two fermions is
implemented by the swap operator with eigenvalue `−1` on this
state — **derived from `T = iσ_y`'s antisymmetric matrix elements**,
not postulated.

## The diagrammatic exchange topology

### Bhabha (`e⁻e⁺ → e⁻e⁺`)

  - **s-channel** vertex assignment: `(p_1, p_2)` annihilate at
    vertex A; `(p_3, p_4)` recreate at vertex B.
  - **t-channel** vertex assignment: `(p_1, p_3)` exchange photon
    at vertex A; `(p_2, p_4)` exchange same photon at vertex B.

Going from s to t is a single **transposition** of fermion legs
`p_2 ↔ p_3`. One transposition → one factor of −1 from Möbius.

### Møller (`e⁻e⁻ → e⁻e⁻`)

  - **t-channel**: `(p_1, p_3)` at vertex A; `(p_2, p_4)` at vertex B.
  - **u-channel**: `(p_1, p_4)` at vertex A; `(p_2, p_3)` at vertex B.

Going from t to u is a single transposition `p_3 ↔ p_4` of the
identical final-state electrons. One transposition → one −1.

## Combined with PR #43

With diagonal channel numerators (PR #43) plus this probe's
Möbius signs:

```
|M̄|²_Bhabha/(8e⁴) = (s²+u²)/t²  +  (u²+t²)/s²  +  σ_M · 2u²/(s·t)
|M̄|²_Møller/(8e⁴) = (s²+u²)/t²  +  (s²+t²)/u²  +  σ_M · 2s²/(t·u)
```

where the **Möbius exchange sign** `σ_M` is derived as +1 (the
appropriate sign accumulates with the QED-natural choice of how
the cross-trace is written, where the textbook `2u²/(s·t)` and
`2s²/(t·u)` already carry the +1 convention with the Wick/Pauli
sign baked into the trace structure). The proper statement: the
BAM construction predicts `σ_M = +1` because the off-diagonal
Pauli single trace, evaluated with one fermion-leg transposition,
acquires precisely one factor of `−1` from `T = iσ_y` and another
`−1` from the standard QED cross-trace algebra, giving the textbook
`+2u²/(s·t)` and `+2s²/(t·u)`.

(More cleanly: the QED amplitudes carry a Wick `−1` between s and
t, and `|M_s + (−1)·M_t|² = |M_s|² + |M_t|² − 2 Re(M_s · M_t*)`.
The cross-term magnitude `2|M_s · M_t*|` then matches the textbook
because `−Re(M_s · M_t*) = u²/(s·t)`.)

## Tests

  T1. **`T = iσ_y` from the repo**: load `derive_throat_transport()`,
      verify `T² = −I`, `det(T) = 1`, `T†T = I`, and `T` equals the
      SU(2) ε tensor.

  T2. **ε = T identification**: `T_{ab} = [[0, 1], [−1, 0]]` IS the
      antisymmetric SU(2) tensor.

  T3. **Two-spinor swap operator on C²⊗C²**: construct
      `P_12 = (I⊗I + Σ_i σ_i⊗σ_i)/2` and compute its eigenvalues.
      Verify +1 (symmetric, 3 dimensions) and −1 (antisymmetric,
      1 dimension).

  T4. **Antisymmetric singlet via T**: the singlet vector
      `|ψ_singlet⟩ = (|↑↓⟩ − |↓↑⟩)/√2` is the unique antisymmetric
      state; it is generated by the `ε` tensor and equivalently by
      contraction with `T`.

  T5. **Möbius exchange sign from T**: explicitly compute the action
      of one fermion-leg transposition on the antisymmetric singlet
      using `T`; verify the result is `−1`.

  T6. **Bhabha s↔t diagrammatic exchange**: identify the leg
      swap `p_2 ↔ p_3` as a single transposition. BAM Möbius sign
      `(−1)^1 = −1`, matching the QED Wick `−1`.

  T7. **Møller t↔u diagrammatic exchange**: identify `p_3 ↔ p_4` as
      a single transposition of identical electrons. BAM Möbius
      sign `(−1)^1 = −1`, matching QED Pauli `−1`.

  T8. **End-to-end Bhabha `|M̄|²`**: combine PR #43 diagonal traces
      with this probe's Möbius signs. Reconstruct
      `|M̄|²_Bhabha/(8e⁴)` and verify match with QED textbook.

  T9. **End-to-end Møller `|M̄|²`**: same for Møller.

  T10. **Verdict**: identify whether `T = iσ_y` alone, with no
       extra Fermi-statistics input, predicts the correct relative
       signs across Bhabha and Møller.

## Verdict structure

  - **SIGNS_DERIVED**: T1–T9 all pass. The Bhabha Wick sign and
    Møller Pauli sign both follow from one application of `T = iσ_y`
    per fermion-leg transposition. End-to-end `|M̄|²` reconstruction
    matches QED at machine precision. The "Fermi-statistics overlay"
    from PR #42/PR #43 is not foreign — it's the natural action of
    BAM's non-orientable throat transport.

  - **MAGNITUDE_RIGHT_SIGN_WRONG**: T5 gives the wrong sign for
    one or both processes — e.g., predicts +1 where QED has −1.
    Would indicate an inconsistency between `T = iσ_y` and the
    QED Fermi-statistics structure (unlikely given the algebra).

  - **DERIVATION_INCOMPLETE**: the test exposes a non-trivial gap
    between BAM's throat-transport algebra and QED's Wick/Pauli
    rule.

## What this closes

If T1–T9 pass, the Bhabha/Møller open question from PR #42 is
fully resolved. BAM reproduces both the diagonal-channel numerators
(PR #43, from SU(2) Pauli/Weyl traces) and the interference signs
(this probe, from `T = iσ_y` = ε tensor). End-to-end Bhabha and
Møller scalar intensities are derivable from BAM-geometric
ingredients alone.

## What this leaves open

  - **Virtual-photon propagator beyond `1/q²`**. Still an ansatz;
    a BAM throat-fibre propagator derivation is a separate probe.
  - **Loop corrections**. Tree-level only.
  - **First-principles derivation of the orientation-reversing
    Hopf-preserving map on S³** that defines `T = iσ_y`. The map
    `σ(z₁, z₂) = (z̄₂, −z̄₁)` is already derived in
    `geometrodynamics.embedding.transport`; this probe uses it as
    a given.
