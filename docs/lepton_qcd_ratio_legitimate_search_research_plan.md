# The legitimate search: does any fit-independent, §8-stable bulk quantity select √σ/m_e ≈ 830? (PR #108)

PR #106 isolated `√σ/m_e ≈ 830.3` as the **single UNDERIVED dimensionless
residual** whose derivation would collapse BAM to one foundational anchor
(the bulk-gravity scale `G`). PR #107 killed the tempting candidate
`N_q + ΔN = 832 ≈ 830`: it merely recycles the phenomenological compensator
`n_part` (`832 = 4·n_part − 4·k_5²`), §8-drifts 764–920, and rides on no
independent integral. PR #107 closed by naming the **only legitimate route
left**: an INDEPENDENT, §8-stable bulk quantity that selects ~830.

This probe runs that search honestly. **It comes up empty.**

## The bar: four criteria

A candidate counts as a *derivation* of the ratio only if it satisfies all
four:

| | criterion | meaning |
|---|---|---|
| C1 | fit-independent | built from fixed geometry, not `n_part` / the v3 fit (#107's failure) |
| C2 | §8-stable | invariant under the `quark_axioms` §8 ablations |
| C3 | precise | within ≈1% of 830.3 |
| C4 | principled | no ad-hoc multiplicative fudge factor |

The key realisation: **C2 is automatic** for any genuine geometric
candidate. Built from BAM's fixed structural quantities — `k_5 = 5`,
`β_lepton = 50π`, the closure constant `2π` — such a candidate never touches
the quark-mass ablations that made `n_part` drift 764–920. So §8-stability,
the bar that killed #107, is free here. The binding constraints are **C3 ∧
C4**: precision *and* principle, together. No candidate clears both.

## The principled scan (best is 5.4% off)

Candidates from `{k_5, β_lepton, 2π}` with small integer powers and no free
factors:

| candidate | value | off% |
|---|---:|---:|
| `2π·k_5³` (= `β_lepton·k_5`) | 785.40 | −5.40% |
| `k_5⁴` | 625.00 | −24.72% |
| `e^(2π)` | 535.49 | −35.50% |
| `2π²·k_5²` | 493.48 | −40.56% |
| `4π·k_5²` | 314.16 | −62.16% |
| `k_5⁵` | 3125.00 | +276.39% |

The best principled quantity, `2π·k_5³ = β_lepton·k_5 = 785.4`, is **−5.4%**
from the target. Nothing built from fixed BAM geometry lands within 1%.

## Every sub-% match is a fit, not a derivation

The only quantities that reach <1% require a factor reverse-engineered from
the target:

| candidate | value | off% | ad-hoc factor |
|---|---:|---:|---|
| `π·265` | 832.5 | +0.27% | 265 (not a BAM integer) |
| `(4/3)·k_5⁴` | 833.3 | +0.37% | 4/3 (ad-hoc) |
| `k_5⁵/3.77` | 828.9 | −0.16% | 3.77 (ad-hoc) |

None of 265, 4/3, 3.77 is a fixed BAM structural quantity; each is tuned to
hit 830. A free factor chosen to match the answer **fails C4** — it is a
fit, exactly the failure mode #107 warned against, just dressed in
different clothes.

## The exponential / dimensional-transmutation route also fails

A large ratio could in principle arise by dimensional transmutation,
`830 = e^S` for a geometric action `S`. But `ln(830.3) = 6.72`, and the only
clean geometric action nearby — the closure quantum `2π = 6.28` — is **7.0%
off** (`e^(2π) = 535`, −35%). No principled action exponentiates to 830 to
<1%.

## No bulk integral selects it

The genuine Tangherlini cavity eigenvalue sums are `O(10–350)`:
`Σω²(n=0..5) ≈ 83`, `Σω(n=0..5) ≈ 20`, `Σω²(n=0..9) ≈ 347`. None lands near
830, and no principled truncation selects it. The independent bulk
shell-stress integral #107 asked for **does not exist**.

## The honest conclusion

No candidate clears C3 ∧ C4. The fit-independent search that #107 named as
the only legitimate route comes up empty. Therefore:

  - `√σ/m_e ≈ 830` stays an **UNDERIVED** open dimensionless residual, and
    the failure of *every* principled route now points to it being
    **IRREDUCIBLY residual** — in the same class as `α` and the electron's
    anomalous lightness (the universal flavor puzzle): a pure number the
    geometry does not fix.
  - **BAM does NOT collapse to a single anchor.** It sits at one
    foundational scale (`G`) + this one open ratio + `α` + the flavor
    puzzle. That is the honest floor.

PR #107's caution is vindicated: having rejected the `n_part` recycling, the
fit-independent search that would have legitimately closed the gap turns up
nothing.
