# Is the quark hierarchy QCD-RG or the flavor puzzle? — refining #97 (PR #98)

PR #97 diagnosed the quark inter-generation hierarchy as "dynamical
(QCD-RG)" — irregular, the program's one non-geometric sector, with
`n_part = 233` compensating it. This probe takes the first concrete step
on that "right route" and, in doing so, **tests the diagnosis**. The
result sharpens it: the hierarchy is dynamical, but the mechanism is not
`α_s` running — it is the **flavor puzzle** (the Yukawa couplings).

## The test: quark mass ratios are RG-invariant

QCD's mass anomalous dimension `γ_m(α_s)` is **flavor-universal** — the
same function for every quark. So under renormalisation-group running

```
m_i(μ₂)/m_i(μ₁) = [α_s(μ₂)/α_s(μ₁)]^{γ₀/β₀}
```

the factor is identical for all flavors and cancels in any ratio:
`m_i(μ₂)/m_j(μ₂) = m_i(μ₁)/m_j(μ₁)`. Quark mass *ratios* are therefore
RG-invariant — `α_s` running sets the overall scale, NOT the hierarchy.
So the irregular hierarchy is **not** generated by QCD running; PR #97's
"QCD-RG" attribution was too specific.

## What it actually is: the flavor puzzle

If the hierarchy is not a running effect, it is set by the Yukawa
couplings at the fundamental scale — free inputs in the Standard Model,
the **flavor puzzle**. The irregularity PR #97 measured (`m_c/m_u ≈ 588`
vs `m_t/m_c ≈ 136`; up/down asymmetric) is exactly its signature: the
Yukawas are hierarchical and irregular for reasons no current theory —
geometric or otherwise — derives. A universal open problem, not a
BAM-specific failing.

## Why leptons fit but quarks overflow

The charged-lepton hierarchy (also a flavor puzzle, `m_τ/m_e ≈ 3477`) is
fit to sub-percent by the BAM closure-ledger winding ladder
(`k ∈ {1,3,5}`, PR #71): the winding quantum number plus the closure
machinery has the dynamic range. The quark shell overtones
(`k=0`, `n=3,4,5`; PR #77/#83) have `ω(1,n) = 3.82, 4.76, 5.70` — a mass
range of only **×1.49** — whereas the up-type quarks span `m_t/m_u ≈
8×10⁴`. The quark shell's geometric capacity is overflowed by ~×5×10⁴; so
the quark Yukawas exceed the compressed shell-overtone range, and
`n_part` is the compensator for that overflow.

## Refinement of #97 (not overturning)

PR #97's core stands: the quark hierarchy is dynamical / non-geometric,
the program's one such sector, and `n_part` can only compensate it. The
refinement is the mechanism — not `α_s` running (ratios are RG-invariant)
but the flavor puzzle (irregular Yukawas overflowing the compressed shell
capacity). More precise, and it locates the quark hierarchy in a problem
open across all of physics, not just BAM.
